# ⚡ Code Crash

A modern full-stack authentication and login system built using cutting-edge technologies like **Next.js 15**, **Prisma ORM**, **NeonDB**, and styled with **Tailwind CSS** and **Lucide Icons**. Secure, scalable, and beautiful.
